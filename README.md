# Eryn-Y.github.io

Project Name: 
Fight ChatGPT 

Project Description: 
ChatGPT is a language model that can simulate real human conversations. Recently, it has also been proven that ChatGPT can write essays that can pass the plagiarism test. It is a threat to academic honesty and intellectual property. Some recent work has successfully detected AI plagiarism https://www.npr.org/2023/01/09/1147549845/gptzero-ai-chatgpt-edward-tian-plagiarism. It is shown that features like “perplexity” (the complexity of text AI was familiar with ) and “burstiness”  (variations of sentences, for example, length) can help determine AI plagiarism. This project aims to use neural networks to determine AI plagiarism or identify features that are similar to “perplexity” and “burstiness”.

Goals: 
Use ChatGPT to generate essays with popular topics (such that we also have essays written by humans under the same topic). 

Label those data 
Train neural networks on those data and see if the neural network can determine AI plagiarism. 

Fine-tune the neural network with features like “perplexity” or “burstiness” to see if it can improve the performance
